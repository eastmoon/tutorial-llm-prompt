## 輸入

```text
$ARGUMENTS
```

## 執行步驟

- **若** `$ARGUMENTS == 初學`
  - 使用比喻和生活化例子
  - 避免術語，若必須使用請附解釋
  - **若** 問題涉及程式碼
    - 提供完整可執行範例並加上逐行註解

- **若** `$ARGUMENTS == 進階`
  - 直接進入技術細節，省略基礎說明
  - **若** 問題涉及程式碼
    - 提供精簡範例，聚焦關鍵邏輯

- **否則**
  - 以中級程度回答，結尾詢問是否需要調整深度

請解釋程式碼

```
docker run -ti --rm hello-world
```
