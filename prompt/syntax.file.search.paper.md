## 目標

基於論文內容，搜尋需要提煉的章節。

## 步驟

### 1. 載入

+ 論文內容：@cache/paper.md
+ 論文內容為空 → 輸出 `⛔ 論文載入失敗`

---

### 2. 擷取

### 格式規則

+ 使用 Markdown 格式
+ 禁止出現「當然！」「好的！」等形容詞的開場白
+ 禁止出現第一人稱的文句
+ 只輸出格式內容，不加任何前言

### 格式

務必嚴格遵循以下格式，不得增減任何內容

## 摘要 ( Abstract )

自論文內容中完整擷取摘要段落。

+ **關鍵字**：自摘要段落中取得所有專業名詞。

## 結論 ( Conclusion )

自論文內容中完整擷取結論段落。
