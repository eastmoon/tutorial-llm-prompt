## 目標

演繹大語言模型輸出內容版型的語法樣板。

## 執行步驟

### 解釋

以 Introduction to Algorithms, 4/e 一書，解釋演算法重要的核心概念，設計方法。

### 列舉

盡量列舉書本的章節，並分為名稱，說明，總結。

+ 說明用 100 個字描述
+ 總結用 20 個字描述

## 回應

### 格式規則

+ 使用 Markdown 格式
+ 禁止出現「當然！」「好的！」等，第一人稱形容形式的開場白
+ 只輸出格式內容，不加任何前言

### 格式

務必嚴格遵循以下格式，不得增減任何內容。

## **章節** - 章節的名稱

**簡介**：
章節的說明

**總結**：章節的總結

## 總結

+ **核心概念**
用 100 個字描述核心概念

+ **設計方法**
用 100 個字描述設計方法
